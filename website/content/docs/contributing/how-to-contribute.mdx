# How to Contribute

APoP is an open-source project and we welcome contributions from the community.

## Ways to Contribute

- **Report bugs** — Open an issue on GitHub
- **Suggest features** — Start a discussion on GitHub Discussions
- **Write documentation** — Improve docs, tutorials, and examples
- **Submit code** — Fix bugs or implement features via pull requests
- **Write examples** — Create industry-specific policy templates
- **Build integrations** — Create middleware for new frameworks
- **Spread the word** — Blog posts, talks, social media

## Getting Started

1. **Fork** the [spec repository](https://github.com/agent-policy-protocol/spec)
2. **Clone** your fork locally
3. **Create a branch** for your changes
4. **Make your changes** and test them
5. **Push** and open a Pull Request

## Development Setup

```bash
git clone https://github.com/YOUR-USERNAME/spec.git
cd spec

# Node.js SDK
cd sdk/node
pnpm install
pnpm test

# Python SDK
cd sdk/python
pip install -e ".[dev]"
pytest
```

## Code of Conduct

We follow the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/). Please be respectful and constructive in all interactions.
